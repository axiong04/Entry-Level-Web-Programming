Web History

20 important internet inventions

1965 - First email program at MIT
1969 - Arpanet is the first internet that runs on packets
1969 - UNIX is used to land on the moon
1969-1977 - Arpanet networks linked together at four universities
1978 - MUD (multi-user dungeon) created
1983 - ip addresses first used
1983 - America online made the internet popular
1985 - domain names and domain name servers
1988 - Morris worm: first internet worm
1989 - WWW created
1991 - First web page
1991 - html created to make web pages
1993 mosaic released, first web browser
1995 - ebay and amazon founded
1995 - java and javascript created
1996 - hotmail created
1998 - google founded
1995-2000 - .com bubble
2004-2006 - facebook, youtube, twitter founded
2007 - iphone
